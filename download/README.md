# 🎮 Эфирная Сага — Aetherial Saga

3D RPG с аниме-стилистикой в открытом мире магии. Создано с любовью.

## 📥 Как скачать и установить

### Windows (.exe)
1. Перейди в раздел **Releases** (сбоку справа)
2. Скачай `Aetherial-Saga-Setup.exe`
3. Запусти установщик и следуй инструкциям
4. Играй!

### macOS (.dmg)
1. Перейди в раздел **Releases**
2. Скачай `Aetherial-Saga.dmg`
3. Открой DMG-файл и перетащи игру в Applications
4. При первом запуске: Правый клик → Открыть (обход Gatekeeper)

### Linux (.AppImage)
1. Перейди в раздел **Releases**
2. Скачай `Aetherial-Saga.AppImage`
3. `chmod +x Aetherial-Saga.AppImage`
4. `./Aetherial-Saga.AppImage`

## 🎮 Управление

| Клавиша | Действие |
|---------|----------|
| WASD / Стрелки | Движение |
| Мышь | Камера (клик для захвата) |
| ЛКМ | Ближний удар |
| ПКМ | Магическая атака |
| 1-5 | Смена стихии (Огонь/Лёд/Молния/Ветер/Земля) |
| Shift | Бег |
| Пробел | Уклонение |
| E | Взаимодействие / Продолжить диалог |
| Tab | Инвентарь |
| Q | Исцеление |
| Escape | Пауза |

## 🌍 Мир игры

- **3 биома**: Зачарованный Лес, Кристальная Пустыня, Земля Вечного Мороза
- **10 NPC**: дружелюбные и враждебные
- **5 стихий магии**: Огонь, Лёд, Молния, Ветер, Земля
- **Система боя** в реальном времени (в стиле Genshin Impact)
- **Кастомизация персонажа**: цвет волос, глаз, одежды, рост
- **Инвентарь и экипировка**
- **Система сохранений** (до 3 слотов)

## 🛠 Технологии

- React Three Fiber (Three.js) — 3D рендеринг с аниме-шейдингом
- Next.js 16 — фреймворк
- Electron — десктопное приложение
- Zustand — управление состоянием
- Tailwind CSS + Framer Motion — UI и анимации
- Simplex Noise — процедурная генерация мира

## 📝 Заметки

- Минимальные требования: любая видеокарта с поддержкой WebGL, 4 ГБ ОЗУ
- Игра полностью оффлайн — не требует интернета
- Рекомендуемое разрешение: 1280×800 и выше
